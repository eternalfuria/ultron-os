# ULTRON OS
Sistema de IA modular inspirado en JARVIS y Ultron.

## Instalación
Linux:
curl -L https://raw.githubusercontent.com/EternalFuria/ultron-os/main/install_ultron_universal.py -o install_ultron_universal.py && python3 install_ultron_universal.py

Windows (PowerShell):
Invoke-WebRequest -Uri "https://raw.githubusercontent.com/EternalFuria/ultron-os/main/install_ultron_universal.py" -OutFile "install_ultron_universal.py"; python install_ultron_universal.py
